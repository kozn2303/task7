# Летящий дрон

Небольшая аркадная игра на **Python** с использованием **Tkinter** и **Pygame**. Игрок управляет дроном, который должен как можно дольше удерживаться внутри безопасного коридора, не касаясь его границ.

---

## Описание проекта

Приложение состоит из двух частей:

* **GUI на Tkinter** — позволяет настроить ширину безопасного коридора перед началом игры;
* **игровое окно на Pygame** — отображает дрона, движущийся коридор, таймер и анимацию столкновения.

Игра была разработана как учебный проект по теме создания графического интерфейса, игровой логики и анимации средствами Python.

---

## Правила игры

Перед началом игры на вкладке управления выбирается **ширина безопасного коридора** с помощью ползунка. После нажатия кнопки **«Старт»** открывается игровое окно.

Игрок управляет дроном в виде небольшого круга:

* при нажатии **Пробела** дрон получает импульс вверх;
* без нажатия дрон плавно опускается вниз под действием силы тяжести.

На экране отображаются **верхняя и нижняя границы безопасного коридора**, которые визуально движутся, создавая эффект полёта.
Задача игрока — **удерживать дрон внутри коридора как можно дольше**.

Если дрон касается верхней или нижней границы:

* происходит **столкновение**;
* запускается **анимация взрыва**;
* игра завершается;
* выводится **время выживания в секундах**.

## Возможности

* графический интерфейс с **двумя вкладками**;
* настройка ширины безопасного коридора;
* управление дроном с помощью клавиши **Space**;

* анимация взрыва при столкновении;
* отображение времени выживания.

---

## Управление

| Клавиша / элемент  | Действие                             |
| ------------------ | ------------------------------------ |
| **Space**          | Поднять дрон вверх                   |
| **Ползунок Scale** | Изменить ширину безопасного коридора |
| **Кнопка «Старт»** | Запустить игру                       |

## Используемые технологии

* **Python 3**
* **Tkinter** — графический интерфейс
* **Pygame** — игровая логика, анимация и отрисовка

## Структура программы

Основные классы проекта:

* **`GameApp`** — создание окна, вкладок интерфейса и запуск игры;
* **`Drone`** — логика движения дрона, гравитация и взлёт;
* **`Corridor`** — построение безопасного коридора и движение стен;
* **`Explosion`** — анимация взрыва при столкновении.

## Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone <ссылка_на_репозиторий>
cd <папка_проекта>
```

### 2. Установить зависимости

```bash
pip install pygame
```

> `tkinter` обычно входит в стандартную установку Python.

### 3. Запустить программу

```bash
python main.py
```

## Пример игрового процесса

1. Открыть приложение.
2. На вкладке **«Главное меню»** выбрать ширину коридора.
3. Нажать **«Старт»**.
4. Управлять дроном клавишей **Space**.
5. Стараться как можно дольше не касаться границ коридора.
6. После столкновения посмотреть результат по времени выживания.
